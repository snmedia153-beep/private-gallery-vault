# 테스트 체크리스트

## 보관함

- 새 보관함 생성
- 마스터 비밀번호 오류 처리
- 잠금 해제 후 메인 화면 진입
- 비밀번호 변경
- 자동 잠금
- 즉시 잠금 단축키

## 파일 가져오기

- 이미지 파일 가져오기
- 영상 파일 가져오기
- 문서/압축 파일 가져오기
- 깨진 유니코드가 포함된 파일명 가져오기
- 중복 파일 가져오기 확인창
- 여러 파일 드래그 앤 드롭

## 탐색과 정리

- 주제 생성, 이름 변경, 삭제
- 주제 검색
- 사이드바 주제 정렬: 최신 등록 순, 이름 내림차순, 이름 오름차순, 사용자 설정 순서
- 파일 검색과 종류별 필터
- 그리드/리스트 전환
- 페이지 이동

## 미디어 보기

- 이미지 더블클릭 열기/닫기
- 영상 더블클릭 열기/닫기
- 영상 재생 중 창 닫기 후 메인 화면 조작
- 영상 재생 중 즉시 잠금
- 잠금 해제 후 앱 유지

## 관리 기능

- 여러 파일 선택 후 주제 변경
- 파일 순서 편집
- 위치 번호로 순서 이동
- 태그 관리
- 중복 관리
- 활동 로그 확인
- 백업 파일 생성
