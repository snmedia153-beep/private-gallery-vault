# 프로젝트 개요

Private Gallery Vault는 개인용 미디어 파일을 로컬에서 암호화하여 보관하고, 주제별로 탐색할 수 있도록 만든 Windows 데스크톱 애플리케이션입니다.

## 설계 방향

- 서버 의존성을 줄인 로컬 우선 구조
- 파일 원본 암호화 저장
- 보기나 외부 열기 시에만 임시 복호화 파일 생성
- WPF 기반의 관리형 데스크톱 UI
- 대량 파일을 다루기 위한 검색, 정렬, 페이지네이션 구조
- 민감한 화면을 빠르게 숨기는 즉시 잠금 흐름

## 주요 모듈

- `VaultService`: 보관함 생성, 잠금 해제, 마스터 키 관리
- `MediaVaultService`: 파일 가져오기, 암호화 저장, 내보내기, 삭제
- `DatabaseService`: SQLite 스키마와 미디어/주제/태그/로그 데이터 접근
- `ThumbnailService`: 이미지 및 영상 썸네일 생성
- `TempFileService`: 임시 복호화 파일 정리
- `MainWindow`: 보관함 탐색, 검색, 필터, 정렬, 주제 관리
- `ViewerWindow`: 이미지/영상 보기와 미디어 관련 작업

## 구현 포인트

- WPF/XAML 기반의 복합 UI 구성
- AES-GCM 기반 파일 암호화 흐름
- SQLite 마이그레이션 및 로컬 데이터 관리
- 드래그 앤 드롭, 멀티 선택, 다중 파일 일괄 작업
- 미디어 플레이어와 임시 파일 핸들 정리 안정화
- 검색, 정렬, 로딩, 잠금 흐름의 데스크톱 UX 개선
